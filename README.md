# P-gina-IMC
Este proyecto consiste en una pagina web desarrollada en Java EE que permite a los usuarios calcular su Índice de Masa Corporal (IMC). El sistema permite que los usuarios se registren, inicien sesión y puedan calcular su IMC ingresando su peso corporal para que cada cálculo realizado se guarde en la base de datos junto con la fecha para que el usuario pueda consultar su historial de registros.

Lo que se utilizó:

- Java EE
- Servidor GlassFish
- Base de datos JavaDB (Derby)
- Maven
- JSP y Servlets
- Servicios Web REST
